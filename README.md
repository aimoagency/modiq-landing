# Modiq 랜딩페이지

한국 모델 에이전시를 위한 부킹·정산 관리 시스템 Modiq의 랜딩페이지.

자립형 정적 사이트입니다 — `index.html`과 `assets/`만으로 동작하며, 빌드 과정이 없습니다.

## 구조
- `index.html` — 랜딩페이지 (반응형, 다크 + 라임 브랜드)
- `assets/` — 대시보드·모바일 스크린샷, 컴카드 데모 영상, 로고

## 배포 (Vercel)
정적 사이트로 자동 인식됩니다. 프레임워크 프리셋 "Other", 빌드 명령 없음, 출력 디렉터리 루트.
`main`에 push하면 Vercel이 자동 배포합니다.
